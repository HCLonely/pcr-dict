## pcr-dict

> 公主连结Re:Dive 中文词库。

### 使用

- 到[releases](https://github.com/HCLonely/pcr-dict/releases)页面下载词库，解压后导入即可。

### 数据来源

- 角色&别名&公会&声优: [HoshinoBot](https://github.com/Ice-Cirno/HoshinoBot)
- 装备&专武: [公主连结WIKI](https://wiki.biligame.com/pcr/%E9%A6%96%E9%A1%B5)

### 词库转换
